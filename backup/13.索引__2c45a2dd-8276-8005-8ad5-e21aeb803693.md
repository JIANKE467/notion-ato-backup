
作用：索引是加速查询的


没有索引查询需要全表扫描（慢）


有索引只需要查询索引即可（快）


| 优点    | 缺点                          |
| ----- | --------------------------- |
| 降低io  | 占空间                         |
| 降低cpu | 降低了更新表速度（改数据还得改索引，ps：不用手动改） |
