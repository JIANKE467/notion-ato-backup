## 分离个十百位

# 分离个十百位


```c
ge = number % 10;         // 个位
shi = (number / 10) % 10; // 十位
bai = number / 100;       // 百位
```


其中number为待分离的正整数


ge为个位数


shi为十位数


bai为百位数

## 将输入字符存入数组

# 将输入字符存入数组


思路

1. 定义两个变量，一个作为暂存用户输入值的变量（x），一个作为数组下标值的变量（i）
2. 用这种方式存入字符至数组

    ```c
    scanf("%d", &array[i]);
    ```

3. 数组下标加1
4. 循环2 3步

    示例：


    ```c
    for (i = 0; i < n; i++)
    {
      printf("Element %d: ", i + 1);
      scanf("%d", &array[i]); // 将用户输入的整数存储在数组中
    }
    ```


## 满十进一（满几进一可类比）

# 满十进一（满几进一可类比）


利用for循环嵌套：(三层for分别控制 个 十 百 位）


例子：打印出100到999的数（体现满10进一的思想）


```c
#include<stdio.h>
int main()
{
 int bai,shi,ge,x;
 for(bai = 1; bai <= 9; ++bai)
   { // 百位数字从1到9
   for(shi = 0; shi <= 9; ++shi)
     { // 十位数字从0到9
     for(ge = 0; ge <= 9; ++ge)
       { // 个位数字从0到9
        x=bai*100+shi*10+ge;
        printf("%d\n",x);
       }
     }
   }
return 0;
}
```
